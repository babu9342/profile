# Naveen Kumar - Premium Portfolio

A modern, responsive, recruiter-friendly personal portfolio website built with pure HTML5, CSS3, and JavaScript.

## Features

- Premium dark theme with glassmorphism cards
- Smooth scrolling and active navbar highlight
- Animated typing effect in hero section
- Scroll reveal animations
- Animated counters and skill progress bars
- Modern project cards and timeline-based education section
- Professional contact section with responsive form
- Back-to-top button and custom scrollbar
- Mobile-first and fully responsive layout

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Folder Structure

```text
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
│   ├── images/
│   ├── icons/
│   └── resume/
└── README.md
```

## Usage

1. Place your profile photo inside `assets/images/` and update image path in `index.html`.
2. Place your resume PDF inside `assets/resume/` as `resume.pdf`.
3. Update social links, contact details, college details, and project links.
4. Open `index.html` in any modern browser.

## Customization Notes

- Colors and gradients can be changed in `:root` variables inside `css/style.css`.
- Typing words can be edited in `words` array inside `js/script.js`.
- Counter numbers and skill percentages can be adjusted directly in `index.html`.
